## Contents of plugins directory

This directory aims at collecting user plugins for the Imago software.

Plugins should be distributed as jar files, and contain a property file that will be parsed 
by the Imago application at startuo to initialize the "Plugins" menu.

Plugins may be stored directly within the "plugins" directory, or within common 
sub-directories:

* Users
* Filters
* Analyze
* IO
* ...
